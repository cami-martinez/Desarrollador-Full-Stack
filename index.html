Desarrollo FrontEnd

Es un termino que hace referencia a la escritura de codigo del lado del cliente. Corresponde a toda caracteristica que puede llevar a cabo el navegador sin intervención del servidor, como Estructura de la pagina, diseño de UI / UX y algunas funcionalidades basicas. Dentro de los lenguajes que componen el FrontEnd se encuentran los siguientes.

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Language)

Lenguaje de marcado que utiliza etiquetas para indicar la funcion que cumple cada elemento, estas tambien poseen atributos que definen las caracteristicas de las mismas.
Las etiquetas HTML se pueden clasificar segun sus caracteristicas en las siguientes categorias

  • De Bloque: son aquellas que ocupan todo el ancho de su contenedor, generan slato de linea y se les puede dar tamaño. Funcionan principalmente como contenedores.
  • De Linea: son aquellas que ocupan el ancho de su contenido, comparten espacio con otros elementos de su tipo, y Funcionan principalmente como contenido.

CSS

Las hojas de estilos en cascadas corresponden a un Lenguaje de Diseño Grafico que utiliza Selector y reglas para definir la apariencia de los elementos que componen nuestra pagina web. Este lenguaje se puede utilizar de diferentes maneras, entre sus implementaciones se encuentran los isguientes tipos de estios:

  • En Linea: el codigo css se escribe dentro de las etiquetas, especificamente dentro del atributo stryle="..."
  • En Bloque: el codigo css se encuentra separado del cogido HTML, con su propia estructura de reglas. En este caso, se puede escribir el mismo dentro del archivo HTML o en un archivo Externo.
