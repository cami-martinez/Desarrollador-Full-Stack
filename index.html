Inicio

Conocimientos

  • HTML Lenguaje de marcado de Hipertexto utilizado para estructurar las paginas web, utiliza etiquetas y atributos para definir la funcion que cumple cada elemento dentro de la pagina.
  • CSS Lenguaje de Hojas de Estilo que facilita el diseño grafico de una pagina web, utiliza selectores y reglas para definir la apariencia y disposicion de los elementoas renderizados.
  • JS Lenguaje de programacion interpretado por el navegador, es de tipado dinamico y se utiliza para la interaccion entre los elementos de la pagina, asi como tambien para realizar peticiones o consultas al servidor.
  • PHP Lenguaje de programacion cuyo objetivo inicial era permitir el desarrollo de paginas web dinamicas, permitiendo al servidor enviar codigo html tras haber ejecutado una funcion o metodo.
  • GIT Sistema de control de versiones que permite realizar un seguimiento a cualquier proyecto que estemos realizando, facilitando el deshacer o repetir cambios en el mismo, como si de un historial se tratase.

Herramientas

  • Bootstrap Framework de CSS desarrollado por el equipo de twitter para facilitar la creacion de sitios web responsive, mediante la implementacion de clases creadas a partir de la metodologia BEM.
  • SASS Preprocesador de CSS que utiliza Ruby como lenguaje de compilacion, que agiliza bastante la creacion de hojas de estilos robustas gracias a caracteristicas como variables, mixins, funciones y bucles.
  • NodeJS Entorno en tiempo de Ejecucion basado en el motor V8 de Google, implementado principalmente para la ejecucion de JS del lado del servidor y desarrollo de aplicaciones de red altamente escalables.
  • ReactJS Biblioteca de Javascript desarrollada por el equipo de facebook diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones de una sola pagina (SPA).
  • MongoDB Sistema de Base de datos de tipo NoSQL orientado a documentos, donde en lugar de utilizar tablas para almacenar datos, los distribuye en colecciones de archivos, utilizando una sintaxis conocida como BSON.
