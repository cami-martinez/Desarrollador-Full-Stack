Formularios HTML

Son herramientas que permiten el intercambio de datos entre cliente y servidor, para utilizarlos dentro de la pagina, simplemente debemos colocar la etiqueta <form> y dentro de la misma los elementos que seran procesados para el envio de los datos.

¿Comó funcionan?

A la etiqueta mencionada anteriormente debemos especificarle por lo menos 2 atributos, que son action="..." el cual indica que debe suceder al enviar el formulario; y method="..." que indica como se debe realizar el intercambio de los datos.

  • GET: metodo cuya prioridad radica en la respuesta del servidor, ya que los datos se envian en texto plano y permanecen visibles en la barra de direcciones. Se utiliza principalmente en Buscadores
  • POST: metodo cuya prioridad esta en proteger los datos enviados por el usuario, ya que cifra los mismos antes de realizar el envio. Se utiliza en formularios de registro, acceso, contacto, etc...

Son herramientas que permiten organizar la informacion de manera tabular, para incorporarlas dentro de la pagina simplementes debemos utilizar la etiqueta <table> dentro de la misma, crear su estructura respetando el modelo de cajas.

Contenido Multimedia

Se conoce con este nombre a todo elemento audiovisiual que podemos incorporar en nuestra pagina. Antiguamente era necesario instalar algun complemento o plugin de terceros como el famoso flash player pero en la actualidad los navegadores brindan soporte de manera nativa a este tipo de elementos. Tambien podemos corroborar la compatibilidad de nuestro navegador con dichos elementos desde el sitio Can I Use

¿Contenido Embebido?

El termino embed hace referencia a un tipo de contenido que fue compartido con autorizacion del autor original. Normalmente cuando insertamos este tipo de elementos en nuestras paginas, utilizamos la etiqueta <iframe> que permite la carga asincrona de contenido.

¿Multimedia sin plugins?

Como se meciona anteriormente los navegadores actuales cuentan con una API de reproduccion a la que podemos acceder si utilizamos las etiquetas <audio> y <video> que son complatibles con los formatos .mp3, .wav .ogg y .mp4, .webm, .ogv respectivamente. Estos elementos admiten elementos como autoplay, muted, loop, controls, etc...
